## 📌 Descripción

**D&D Textil** es una tienda digital enfocada en la venta de telas, desarrollada como proyecto formativo en el **SENA**. El sistema permite explorar productos, gestionar un carrito de compras y administrar inventarios de forma eficiente.

---

## 🛒 Funcionalidades principales

* **Catálogo:** Exploración de textiles por categorías.
* **Carrito:** Proceso de compra intuitivo.
* **Admin:** Panel de gestión de productos y pedidos.
* **Responsive:** Optimizado para cualquier dispositivo.
